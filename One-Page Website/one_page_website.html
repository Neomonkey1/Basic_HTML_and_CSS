This is what your use article tags for

You use the article tag to specify independent or self-contained content.
It doesn't render the text in any special way on it's own.

Details and Summary

The Details tag enables additional details to be opened and closed.
The Summary tag is normally used in conjunction with the <details> tag,
To specify a visible heading for the Details.

Time and DateTime tags

The time tag is used to define a specific time
the Lunar Hotel takes reservations from 08:00 to 17:00 every weekday.

The DateTime tag is used to format a time that browsers can then offer to add date reminders
through the user's calendar. Also so search engines can produce smarter search results.
